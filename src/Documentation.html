Editor

Mode Switching

  • - Alt + Space = Enter normal mode
  • - a = Enter Insert after letter
  • - i = Enter Insert before letter
  • - A = Enter at the end of the line
  • - I = Enter at the front of the line
  • - o = Enter bellow the line

Movements in editor

  • - h = left
  • - j = bottom
  • - k = top
  • - l = right
  • - f + (any letter) = move to (any letter)
  • - w = move foward a word
  • - b = move backward a word
  • - G = move bottom of file

Spacemacs mode

Window

  • - Space + 1 = Focus First Window
  • - Space + 2 = Focus Second Window
  • - Space + w + / = Split window right
  • - Space + w + - = Split window bellow
  • - Space + w + [ = Window Shrink
  • - Space + w + ] = Window Enlarge
  • - Space + w + d = delete window

Open Function

  • - Space + ' = Open Intergrated terminal
  • - Space + / = Find Word in file
  • - Space + . = Repeat action
  • - Space + Space = Command Mode

File

  • - Space + f f = Find file in workspace
  • - Space + f n = New file
  • - Space + f D = Delete current file
  • - Space + f R = Rename currrent file
  • - Ctrl k + Ctrl o = Open new folder
